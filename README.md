# ui.prompt
一个弹出层的组件
本组件需要导入jquery才可正常使用
<blockquote style="padding:10px;border-left:5px solid #888;background:#eee;">
    <p>特点：</p>
    <ol>
        <li>多弹框层次关系</li>
        <li>按下ESC键关闭顶层窗体</li>
        <li>窗体标题区域可拖拽</li>
        <li>窗体高于网页高度会出现滚动条</li>
        <li>自定义按钮和对应点击事件</li>
    </ol>
</blockquote>
